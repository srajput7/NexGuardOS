About SecureOS

SecureOS is a cross-platform application designed to detect vulnerabilities, enhance security, and configure operating systems for optimal protection. Whether youâ€™re using it for personal use, business, or advanced setups, SecureOS has you covered.

Features

  • Detect open ports to find vulnerabilities
  • Monitor network for packet sniffing attempts
  • Auto-configures OS-specific security features:
    • macOS: FileVault, Firewall, and SIP
    • Linux: AppArmor/SELinux, iptables, SSH hardening
    • Windows: Defender Firewall, BitLocker, RDP hardening
  • Choose your profile: Personal, Business, or Advanced

How to Build Executables (For Developers)

If you're a developer and want to create the executables yourself, follow the steps below to build SecureOS for your platform:

  • Install PyInstaller: pip install pyinstaller
  • Run the following commands to build the executables:
    • For Windows: pyinstaller --onefile --windowed SecureOS.py
    • For macOS/Linux: pyinstaller --onefile SecureOS.py
  • After running the above commands, the executables will be located in the dist folder.
